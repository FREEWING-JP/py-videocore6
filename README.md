# py-videocore6

Python library for GPGPU programming on Raspberry Pi 4.

## Installation

```
$ git clone https://github.com/Idein/py-videocore6
$ cd py-videocore6/
$ pip3 install -r requirements.txt
$ python3 setup.py build
$ sudo python3 setup.py install
```

## Testing

```
$ pip3 install nose
$ nosetests -v -s
```
